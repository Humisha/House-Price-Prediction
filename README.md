# 🏠 House Price Prediction

A Machine Learning project to predict house prices using real-world data.

## 📌 About
Predicts house prices based on area, bedrooms, bathrooms, 
city, furnishing status and more.
Dataset has 2000 records from cities like 
Delhi, Pune, Kolkata, Chennai, Hyderabad.

## ⚙️ Project Workflow
- 📂 Data Loading — Loaded dataset using Pandas
- 🧹 Preprocessing — Handled missing values and outliers
- 🔍 EDA — Explored patterns and visualized data
- ⚙️ Feature Engineering — Encoded and scaled features
- 🤖 Model Training — Trained ML algorithms
- 📊 Evaluation — Measured model performance

## 🤖 Algorithms Used
- Linear Regression

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## 📁 Dataset
- File: enhanced_house_price_dataset.csv
- Records: 2000 rows
- Features: Area, Bedrooms, Bathrooms, Stories, 
  Parking, Age, Furnishing, City and more
